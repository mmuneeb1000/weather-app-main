# Weather App With OpenMateo - Frontend Mentor

A responsive weather application built with React that provides current weather conditions, hourly forecasts, and 7-day forecasts for locations around the world.

## Features

- Search locations with autocomplete suggestions
- Current weather conditions
- Hourly weather forecast
- 7-day weather forecast
- Switch between Metric and Imperial units
- Responsive design for desktop and mobile devices
- Weather icons based on current conditions
- Error handling for invalid searches

## Built With

- React
- JavaScript
- CSS
- Open-Meteo API
- Open-Meteo Geocoding API

## How It Works

1. Enter a city name in the search bar.
2. Select a location from the suggestions.
3. View current weather information.
4. Browse hourly forecasts.
5. Check the 7-day forecast.
6. Switch between Metric and Imperial units at any time.

## Data Provided

The application displays:

- Current temperature
- Weather conditions
- Wind speed
- Humidity
- Daily high and low temperatures
- Hourly forecasts
- 7-day forecasts

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project folder:

```bash
cd weather-app
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
